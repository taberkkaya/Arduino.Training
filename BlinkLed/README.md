## 📝 Description
This lesson covers:
- How to connect an LED to an Arduino
- Writing a simple blink program
- Understanding the delay function and its effect on LED timing

## 🔧 Requirements
- Arduino board (Uno, Mega, etc.)
- LED
- 330Ω resistor
- Jumper wires

Upload the code and watch your LED blink!
